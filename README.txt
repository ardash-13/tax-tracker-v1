Non-VAT Income & Expense Tax Tracker – User Instructions (Released Version)
This application is intended for tracking and estimating taxes.

1. Run the Application

Step 1: Download

Download the ZIP file from the Google Drive link you received.

Step 2: Extract

Locate the downloaded ZIP file on your computer.

Right-click the file and select Extract All…

Choose a location (Desktop is recommended).

⚠ The application will NOT work properly if you do not extract the ZIP file.

Step 3: Run the App

Open the folder you just extracted.

Double-click NonVatTaxTracker.exe to start the program.

2. Data Storage

All your records, backups, and database files are automatically stored in the data folder inside the same folder as the .exe.

Do NOT delete, rename, or move the data folder, as it contains all your saved records.

3. Moving the Application

You may move the entire application folder anywhere on your computer (Desktop, Documents, USB drive, etc.).

Your data remains safe as long as the data folder stays inside the application folder.

❌ Do not move the .exe file alone.
✔ Always keep the .exe and data folder together.

4. Backups

Backups are created automatically inside the data folder.

You do not need to manually create backups unless you want an extra copy outside the application.

5. Updating the Application

If you replace the .exe with a newer version:

Keep the existing data folder in the same directory.

Your records and settings will remain intact.

6. General Usage Tips

Enter income and expenses as needed.

Use filters to view records by year or quarter.

All calculations (gross income/expense, withholding tax, ATC, paid/received amounts) are done automatically.

Undo actions are available if you make a mistake.
